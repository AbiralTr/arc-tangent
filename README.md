# Arc-Tangent

A lightweight end-to-end testing repository for validating and experimenting with the Arc-Web stack using modern browser automation and Python-based testing workflows.

## Features

- Playwright integration with Chromium
- Pytest test runner and fixtures
- Parallel test execution support
- Headless and headed browser modes
- Screenshot and trace capture
- CI/CD ready
- Environment-based configuration
- Async-compatible test patterns

---

## Tech Stack

| Tool | Purpose |
|---|---|
| Playwright | Browser automation |
| Chromium | Browser engine |
| Pytest | Test framework |
| Python 3.11+ | Runtime |
| pytest-asyncio | Async test support |
| pytest-xdist | Parallel execution |
| GitHub Actions | CI automation |
